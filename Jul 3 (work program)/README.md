## โปรแกรมตรวจสอบเลขคู่หรือเลขคี่

ชื่อโปรแกรม
- EvenOdd

Input
- รับค่าตัวเลขจำนวนเต็มจากผู้ใช้ผ่านแป้นพิมพ์

Process
1. รับค่าตัวเลขจากผู้ใช้
2. ตรวจสอบว่าตัวเลขหารด้วย 2 ลงตัวหรือไม่
3. หากหารลงตัว แสดงผลว่า Even number
4. หากหารไม่ลงตัว แสดงผลว่า Odd number

Output
- แสดงข้อความ Even number หากเป็นเลขคู่
- แสดงข้อความ Odd number หากเป็นเลขคี่

วิธีรันโปรแกรม
1. ป้อนตัวเลขที่ต้องการตรวจสอบ แล้วกด Enter
2. โปรแกรมจะแสดงผลว่าเป็นเลขคู่หรือเลขคี่

สิ่งที่ได้เรียนรู้
- การใช้คำสั่ง if-else เพื่อตรวจสอบเงื่อนไข
- การใช้ตัวดำเนินการ % เพื่อตรวจสอบเลขคู่และเลขคี่


## คำนวณคะแนนรวมและตัดสินผลผ่าน/ไม่ผ่าน

ชื่อโปรแกรม
- PassFail

Input
- รับคะแนนสอบกลางภาค
- รับคะแนนสอบปลายภาค

Process
1. รับคะแนนกลางภาคและคะแนนปลายภาคจากผู้ใช้
2. คำนวณคะแนนรวม โดยนำคะแนนทั้งสองมาบวกกัน
3. แสดงคะแนนรวม
4. ตรวจสอบว่าคะแนนรวมมีค่าตั้งแต่ 50 คะแนนขึ้นไปหรือไม่
5. หากคะแนนรวมมากกว่าหรือเท่ากับ 50 ให้แสดงผล Pass
6. หากคะแนนรวมน้อยกว่า 50 ให้แสดงผล Fail

Output
- แสดงคะแนนรวม
- แสดงผลการตัดสินว่า Pass หรือ Fail

วิธีรันโปรแกรม
1. ป้อนคะแนนกลางภาคและคะแนนปลายภาค แล้วกด Enter
2. โปรแกรมจะแสดงคะแนนรวมและผลการตัดสินผ่านหรือไม่ผ่าน

สิ่งที่ได้เรียนรู้
- การคำนวณค่าด้วยตัวแปร
- การใช้คำสั่ง `if-else เพื่อตรวจสอบเงื่อนไข


## ทบทวน Array (คะแนนรวมและค่าเฉลี่ย)

ชื่อโปรแกรม
- ScoreArray

Input
- รับคะแนนจากผู้ใช้จำนวน 5 ค่า

Process
1. สร้างอาร์เรย์ scores ขนาด 5 ช่องสำหรับเก็บคะแนน
2. รับคะแนนจากผู้ใช้ทีละค่า
3. นำคะแนนแต่ละค่ามาบวกสะสมไว้ในตัวแปร total
4. คำนวณค่าเฉลี่ยด้วยสูตร average = total / จำนวนคะแนน
5. แสดงคะแนนรวมและค่าเฉลี่ย

Output
- แสดงคะแนนรวม
- แสดงค่าเฉลี่ยของคะแนน

วิธีรันโปรแกรม
1. ป้อนคะแนนจำนวน 5 ค่า โดยกด Enter หลังจากป้อนแต่ละค่า
2. โปรแกรมจะแสดงคะแนนรวมและค่าเฉลี่ยของคะแนนทั้งหมด

สิ่งที่ได้เรียนรู้
- การใช้งาน Array
- เรียนรู้การใช้ลูป for


## Mini Challenge (โปรแกรมตรวจสอบเลขคู่หรือเลขคี่)

ชื่อโปรแกรม
- CountEvenOdd

Input
- รับตัวเลขจำนวนเต็มจากผู้ใช้จำนวน 10 ค่า

Process
1. สร้างอาร์เรย์ numbers ขนาด 10 ช่อง
2. รับค่าตัวเลขจากผู้ใช้ทีละจำนวน
3. ตรวจสอบว่าตัวเลขแต่ละตัวเป็นเลขคู่หรือเลขคี่ โดยใช้เครื่องหมาย % (Modulo)
4. หากหารด้วย 2 ลงตัว ให้นับเพิ่มใน evenCount
5. หากหารด้วย 2 ไม่ลงตัว ให้นับเพิ่มใน oddCount
6. เมื่อรับข้อมูลครบทั้ง 10 จำนวนแล้ว แสดงผลจำนวนเลขคู่และเลขคี่

Output
- แสดงจำนวนเลขคู่
- แสดงจำนวนเลขคี่

วิธีรันโปรแกรม
1. ป้อนตัวเลขจำนวนเต็มจำนวน 10 ค่า โดยกด Enter หลังจากป้อนแต่ละค่า
2. โปรแกรมจะแสดงจำนวนเลขคู่และจำนวนเลขคี่ที่ป้อน

สิ่งที่ได้เรียนรู้
- เรียนรู้การใช้งาน Array
- เรียนรู้การใช้ลูป for
- เรียนรู้การใช้เงื่อนไข if-else


## ใช้ GenAI ช่วยตรวจโค้ด (Mini Challenge)

Prompt ที่ใช้ถาม AI
- ช่วยตรวจสอบโค้ด Java ต่อไปนี้ว่ามีข้อผิดพลาดหรือไม่ และอธิบายอย่างเข้าใจง่าย และแนะนำว่าไปใช้ได้จริงหรือไม่

1. AI พบข้อผิดพลาดอะไรหรือไม่
ตอบ: ไม่พบข้อผิดพลาดของโค้ด

2. คำแนะนำของ AI ถูกต้องหรือไม่ เพราะเหตุใด
ตอบ: ถูกต้อง เพราะโปรแกรมสามารถทำงานได้จริง

3. นักศึกษาแก้ไขโค้ดตาม AI หรือไม่
ตอบ: แก้ไขบางส่วน โดยเพิ่มคำสั่ง sc.close();

4. นักศึกษาได้เรียนรู้อะไรจากการใช้ AI ตรวจโค้ด
ตอบ: ได้เรียนรู้วิธีตรวจสอบความถูกต้องของโค้ด

5. มีข้อควรระวังอะไรในการใช้ AI ช่วยเขียนโปรแกรม
ตอบ: ควรอ่านและทำความเข้าใจโค้ดที่ AI แนะนำก่อนนำไปใช้ ไม่ควรคัดลอกโดยไม่ตรวจสอบ เพราะ AI อาจให้คำแนะนำที่ไม่เหมาะสมกับโจทย์หรืออาจมีข้อผิดพลาดได้
